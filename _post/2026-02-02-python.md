Post 1. Learning Python for the First Time
Learning Python for the First Time: My 
Introduction to Programming
When I entered Computer Engineering, programming was one of the subjects I was both 
excited and nervous about. Before university, programming looked mysterious to me. I used 
computers every day, but I had no idea how software was actually created or how 
applications understood user instructions.
Programming Fundamentals introduced us to Python, and this became my first real 
interaction with coding.
Initially, even simple concepts felt difficult. Variables, loops, conditional statements, and 
functions all seemed like completely new languages. Sometimes a single missing colon or 
incorrect indentation would stop my program from running. It was frustrating to spend twenty 
minutes searching for an error only to discover a small typing mistake.Over time, I realized 
programming was less about memorizing syntax and more about developing logical thinking. 
Every problem required breaking a larger task into smaller steps that a computer could 
understand.

One thing I appreciated about the course was that it emphasized problem solving rather than 
simply writing code. Under the guidance of Dr. Bilal Ahmad during Programming 
Fundamentals, we were encouraged to think about how software could eventually solve 
practical problems rather than just classroom exercises.
As my confidence improved, Python became less intimidating and more enjoyable. The first 
time my own program worked exactly as intended, I experienced a level of satisfaction that is 
difficult to describe. #MLwithDrBilalAhmad #DrBilalAhmad #MLProject
Looking back now, learning Python was not simply learning a programming language. It was 
learning how engineers communicate with machines and teach them to solve problems.
